## AI Minecraft Builder
---
## 프로젝트 개요

본 프로젝트는 건축 이미지(컨셉아트, 일러스트, 건물 사진)을 학습하여 건축 스타일을 분류하고, 분류 결과를 바탕으로 Minecraft 구조물을 자동 생성하는 시스템을 개발하는 것을 목표로 한다.

기존 Minecraft 맵 제작은 건축 실력과 많은 시간이 필요하지만, 본 프로젝트는 컴퓨터 비전과 딥러닝을 활용하여 이미지로부터 건축 정보를 추출하고 자동으로 구조물을 생성함으로써 맵 제작 과정을 보조한다.

---
## 프로젝트 목표

### 1. 건축 이미지 데이터셋 구축
다양한 스타일의 건축 이미지를 수집하여 데이터셋을 구축한다.

예상 클래스
- Castle
- Dungeon
- village

각 클래스당 약 30~50장의 이미지를 수집한다.

### 2. 건축 스타일 분류 모델 학습
구축한 데이터셋을 사용하여 CNN 기반 이미지 분류 모델을 학습한다.

입력
- 건축 이미지
  
출력
- 건축 스타일 클래스

### 3. 건축 특징 추출
분류 결과를 기반으로 구조 생성에 필요한 건축 특징을 추론한다.

### 4. Minecraft 구조 생성
추출된 특징을 바탕으로 Minecraft 구조 생성 규칙을 적용한다.

### 5. Minecraft 적용
생성된 구조를 Minecraft에서 사용할 수 있는 형태로 저장한다.
